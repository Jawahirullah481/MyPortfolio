Intro

This is a UI clone project of Whatsapp application buit using HTML, CSS and JS. It has all main screens of whatsapp application. It has the UI to send message, view status and make calls.

Preview

Technologies used

  • HTML
  • CSS
  • JS

Features

  • This application clone contains UI of chat screen, status screen and call screen.
  • It is interactive clone. So, If you click chat overview or status views or call logs, you will be directed to respective screens.
  • You can also send messages in chat screen.
  • status will come back after few seconds as in real application.
  • You can press back icon and call off icons to come back from the screens.

Getting started

To get started, clone this repository to your local machine and open the project in your favorite IDE.
